For the selected disease, enter DALY-Weights for each sex and age class

Files of this format can be created using the "DALY weights for disease" worksheet in the "macro/2-Disease/Disease.xml" workbook.

DALY(disability adjusted life years)-weights are measures of disease severity. A value of 0 means no impairment through disease, a value of 100 means a health state as worse as death.
